MARK AND SONS PROJECTS — Static Website
========================================

Pure HTML5 / CSS3 / Vanilla JavaScript. No frameworks, no build tools.

To view:
  Just open index.html in any modern browser.

Folder structure:
  /index.html
  /css/style.css
  /js/script.js
  /assets/images/   (photography)
  /assets/icons/    (SVG icons — all local)
  /assets/videos/   (drop local videos here if needed)
